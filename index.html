프로젝트

프로젝트 1: 수어 번역 및 채팅 클라이언트 개발

설명: MediaPipe와 Socket 통신을 활용하여 실시간으로 수어를 텍스트로 번역하고, 번역된 텍스트를 전송할 수 있는 간단한 채팅 클라이언트 프로그램을 개발했습니다.

사용 기술: Python, OpenCV, MediaPipe, Socket

링크: 깃허브 저장소

프로젝트 2: 딥러닝 기반 상품 이미지 분류 모델

설명: CNN(Convolutional Neural Network) 모델을 활용하여 전자상거래 플랫폼의 상품 이미지를 자동으로 분류하는 시스템을 개발했습니다. 정확도 95%를 달성하여 수동 분류 작업 시간을 80% 단축했습니다.

사용 기술: Python, TensorFlow, Keras, OpenCV

링크: 깃허브 저장소 | 관련 발표자료
